# terraformSimpleVM

A simple deployment of a linux vm in Azure

## Description
Deploys a network with a subnet and a ubuntu 16.04 LTS VM on top.

## Deployment
### Includes
- prefix
- resource group
- network
- subnet
- nic
- pip
- nsg
- vm
- output
